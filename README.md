# Professional Student Portfolio

A modern, responsive portfolio website designed for creative professionals. Features a glassmorphism design, animated backgrounds, and a dynamic project gallery.

## Features

*   **Responsive Design**: Fully responsive layout that works on all devices.
*   **Glassmorphism UI**: Modern frosted glass effects on navbars and modals.
*   **Dynamic Project Gallery**: JS-powered gallery with modal popup details.
*   **Admin Panel**: Manage projects (add/edit/delete) directly from the browser.
*   **Animated Backgrounds**: Smooth CSS gradient shifts.
*   **Contact Form**: WhatsApp integration for direct messaging.

## Tech Stack

*   HTML5
*   CSS3 (Variables, Flexbox, Grid, Animations)
*   JavaScript (ES6+, Data Handling)

## Setup

1.  Clone the repository.
2.  Open `index.html` in your browser.
3.  To use the Admin Panel, navigate to `/admin.html` (Password: `admin123`).
